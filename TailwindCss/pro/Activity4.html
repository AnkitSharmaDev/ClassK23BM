Tailwind CSS Box Model Example

Box Model

This box demonstrates the properties of the box model using Tailwind CSS.

Padding

This box has p-3, which means padding of 1rem (16px) on all sides.

Border

This box has border-2 border-blue-800, which means border of 2px solid blue.

Margin

This box has m-3, which means margin of 1rem (16px) on all sides.

Width & Height

This box has w-64 (16rem and h-32(8 rem).
